CSS Exercise

To start, create a file called style.css in the same directory as this html file. Create a link to it in the of this document. The link should look like this.

<link rel="stylesheet" href="style.css">
Do not copy this text directly, as the < and > symbols may be encoded differently.

You can use tag names, classes, or a combination of the two to select different elements. An example of each might look like this:

/* Targets all paragraphs */
p {
  color: red;
}

/* Targets any element with class="paragraph-bold" */
.paragraph-bold {
  font-weight: bold;
}

/* Targets any paragraph that is a child of an element with class="section-name" */
.section-name p {
  line-height: 1.6;
}

/* Targets both heading-1s and heading-2s */
h1,
h2 {
  margin: 0 0 10px;
}

Typography

To use typefaces from Google Fonts, click the plus icon on the typeface, click the black bar at the bottom of your screen, click @import, copy the code (leaving out the style tags!) and paste it at the top of your CSS file. Google Fonts will show an example of the font-family rule you need to use.

  2. Change the line height of all list items to 1.6.
  3. Make all <small> elements on the page italic.
  4. Change the font-weight of the second paragraph to be bold. Hint: use a class.
  5. Change the font-family of the document to a typeface of your choosing from Google Fonts.
  6. Make all headings a different typeface.
  7. Remove the underline of the link above. The CSS property is text-decoration.
  8. Center the first heading on the page.
